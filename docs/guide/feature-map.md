# 功能地图与 GUI 位置

这页专门给两类场景准备：

- 你想快速知道某个功能在界面的哪里。
- 你想让 AI 读取文档后，能准确定位 Folia 当前实现里的入口、开关和面板。

本文以主仓库 `folia-major` 当前 GUI 实现为准，优先描述“用户在屏幕上实际能看到和点到的东西”。

## 怎么看这页

- `主页 > 左上角 > 设置按钮` 表示从主页左上区域进入。
- `播放页 > 右侧面板 > 控制` 表示先打开播放页右侧面板，再切到 `控制` 标签。
- `设置 > 帮助` 与 `设置 > 选项` 指的是设置弹窗顶部的两个一级标签。
- `桌面版专属` 表示 Web 版通常不会出现，或意义明显弱很多。

## 全局 GUI 总览

| 区域 | 你会看到什么 | 主要作用 |
| --- | --- | --- |
| 主页顶部左侧 | `Folia` 标题、设置/帮助按钮、扫描进度环 | 打开帮助与设置、查看本地曲库扫描状态 |
| 主页顶部中间 | 标签切换条 | 在 `歌单 / 电台 / 专辑 / 本地 / Navidrome / 舞台` 之间切换 |
| 主页顶部右侧 | 搜索框 | 按当前标签搜索网易云、本地曲库或 Navidrome |
| 主内容区 | 海报墙、专辑墙、本地列表、Navidrome 视图 | 浏览和进入具体内容 |
| 播放页中央 | 歌词动画、封面、歌曲信息 | Folia 的核心沉浸式播放体验 |
| 播放页底部 | 播放控制栏 | 播放、暂停、进度、音量 |
| 播放页右侧 | 右侧面板 | 主题、歌词、队列、账号、本地/服务器专属信息 |
| 设置弹窗 | `帮助 / 选项` 双页 | 帮助、快捷键、设置中心、实验功能 |
| 命令面板 | 播放页按 `S` 打开 | 用命令直达页面、面板、设置和动作 |
| 遥控窗 | `Folia Remote` 小窗，桌面版专属 | 不切回主窗口也能控制播放、录制和透明模式 |

## 主页

### 顶部栏

| GUI 位置 | 功能 | 说明 |
| --- | --- | --- |
| 主页 > 左上角 > 设置图标 | 打开设置弹窗 | 默认先进入 `帮助` 页；有更新时按钮会带红点 |

| 主页 > 右上角 > 搜索框 | 搜索当前来源 | 会随当前标签自动切换搜索对象 |

## 播放页

### 主界面

| GUI 位置 | 功能 | 说明 |
| --- | --- | --- |
| 播放页中央 | 歌词动画显示 | 所有 visualizer 都在这里渲染 |
| 播放页中央/上方 | 当前歌曲标题、艺人、专辑 | 是用户识别当前歌曲的主信息区 |
| 播放页底部 | 播放控制栏 | 播放/暂停、上一首、下一首、进度、音量 |
| 播放页右缘 | 右侧面板开关 | 打开后可切换多个标签页 |

### 播放页快捷动作

| GUI 位置或方式 | 功能 | 说明 |
| --- | --- | --- |
| `Space` | 播放 / 暂停 | 帮助页和用户指引都有写 |
| `Ctrl + ←/→` 或 macOS `Cmd + ←/→` | 上一首 / 下一首 | 桌面端歌曲切换快捷键 |
| `←/→` | 快退 / 快进 5 秒 | 直接作用在当前播放 |
| `P` | 打开 / 关闭右侧面板 | 即使隐藏了右侧按钮，也还能这样打开 |
| `H` | 隐藏播放页控件 | 用户指引中列为播放器快捷键 |
| `F11` | 浏览器全屏 | Web 和桌面版都可作为显示层使用 |
| `S` | 打开命令面板 | 播放页最重要的高级入口之一 |

## 播放页右侧面板

### 固定标签

| GUI 位置 | 功能 | 说明 |
| --- | --- | --- |
| 播放页 > 右侧面板 > `封面` | 歌曲信息页 | 点击歌手或专辑可跳转详情；点击标题可复制歌曲信息；如果是已匹配的网易云歌曲，`Ctrl+点击标题` 还能打开对应网易云歌曲页 |
| 播放页 > 右侧面板 > `控制` | 主题、循环、音量、动画模式 | 是播放页内最常用的视觉控制区 |
| 播放页 > 右侧面板 > `队列` | 当前播放队列 | 支持点击任意歌曲跳转播放、打乱队列、把当前队列保存为本地歌单 |
| 播放页 > 右侧面板 > `账号` | 网易云账号与同步 | 显示头像、昵称、会员状态、音质切换、同步数据、退出登录 |

### 按歌曲来源追加的标签

| GUI 位置 | 何时出现 | 功能 |
| --- | --- | --- |
| 播放页 > 右侧面板 > `本地` | 当前歌曲来自本地音乐 | 查看文件名、大小、比特率、歌词来源、ReplayGain；上传歌词文件；在线匹配歌词；切换 `本地 / 内嵌 / 在线` 歌词来源；调节歌词时间偏移 |
| 播放页 > 右侧面板 > `Navidrome` | 当前歌曲来自 Navidrome | 查看服务器歌曲 ID；在线匹配歌词；查看当前使用的是 `服务器歌词` 还是在线匹配结果；调节歌词时间偏移 |
| 播放页 > 右侧面板 > `歌词` | 当前歌曲来自网易云在线播放 | 导入歌词文件；重新在线匹配；在 `在线来源` 与 `导入文件` 之间切换；调节歌词时间偏移 |

### `控制` 标签里能做的事

| GUI 位置 | 功能 | 说明 |
| --- | --- | --- |
| 右侧面板 > `控制` > 第一排按钮 | 循环、收藏、生成 AI 主题 | 分别对应循环模式切换、喜欢歌曲、为当前歌曲生成 AI 主题 |
| 右侧面板 > `控制` > 音量 | 调整当前播放器音量 | 包括静音切换 |
| 右侧面板 > `控制` > 动画模式 | 切换 visualizer | 可直接在这里切到 `流光 / 心象 / 云阶 / 浮名 / 群唱 / 倾诉 / 莫奈` |
| 右侧面板 > `控制` > 动画强度 | 切换 `calm / normal / chaotic` | 是播放页内的快速强度开关，不等于完整调参 |
| 右侧面板 > `控制` > 背景 | 切换默认主题 / AI 主题 / 自定义主题 | 也可在这里快速切换明暗模式和封面主色参与背景 |

## 设置弹窗

设置弹窗顶部有两个一级标签：`帮助` 和 `选项`。

### 帮助页

| GUI 位置 | 功能 | 说明 |
| --- | --- | --- |
| 设置 > `帮助` | 查看键盘快捷键 | 包含主页导航、播放控制等基础快捷键 |
| 设置 > `帮助` > `Show User Guide` | 打开用户指引 | 是一个多页引导，包含即时搜索、播放器快捷键、命令面板、主题说明 |
| 设置 > `帮助` > `Need help?` | 打开 AI 求助提示词弹窗 | 会生成一段附带文档站和 GitHub 仓库链接的提示词，方便复制到外部 AI 工具中求助 |
| 设置 > `帮助` > 底部版本行 | 复制版本信息 | 便于反馈问题时带上版本、分支和提交哈希 |

### 选项页总览

| GUI 位置 | 功能 | 说明 |
| --- | --- | --- |
| 设置 > `选项` > `视觉设置` | 主题、歌词动画、首页布局 | 影响 Folia 的主要观感 |
| 设置 > `选项` > `播放控制` | 队列、歌词源、播放设备 | 影响播放行为而不是视觉样式 |
| 设置 > `选项` > `存储与缓存` | 缓存分类、媒体缓存 | 适合清理空间和调媒体缓存 |
| 设置 > `选项` > `集成设置` | Discord、OBS、Stage、Now Playing、Navidrome | 所有对外连接入口都在这里 |
| 设置 > `选项` > `桌面端设置` | 托盘、更新、AI 提供商 | 仅桌面版真正完整 |
| 设置 > `选项` > `实验室` | 性能、隐藏 UI、歌词过滤 | 放置高影响、偏实验性质的开关 |

### 视觉设置

| GUI 位置 | 功能 | 说明 |
| --- | --- | --- |
| 设置 > `选项` > `视觉设置` > 主题与配色 | 默认主题、AI 主题、自定义主题 | 也可进入 `Theme Park` 做完整主题编辑 |
| 设置 > `选项` > `视觉设置` > `打开 Theme Park` | 进入主题编辑器 | 编辑明暗双主题颜色 |
| 设置 > `选项` > `视觉设置` > `自动应用歌曲已缓存的 AI 主题` | 切歌时自动恢复缓存主题 | 和“优先使用自定义主题”是不同策略 |
| 设置 > `选项` > `视觉设置` > `自动调用 AI 并应用主题` | 歌曲没有缓存主题时自动生成 | 会消耗实际模型 token |
| 设置 > `选项` > `视觉设置` > `播放页透明背景` | 透明播放页窗口 | 适合抠像、叠加和透明桌面玩法 |
| 设置 > `选项` > `视觉设置` > `自动隐藏控制栏` | 自动隐藏进度条和右侧按钮 | 适合更纯净的展示画面 |
| 设置 > `选项` > `视觉设置` > `歌词动画实验台` | 打开可视化实验台 | 对各 visualizer 做完整调参 |
| 设置 > `选项` > `视觉设置` > 首页布局样式 | `经典(旧版)` / `万象(新版)` | `万象` 才会继续出现 3D 网格卡片样式 |
| 设置 > `选项` > `视觉设置` > 3D 网格卡片样式 | `纯图片封面` / `拍立得卡片` | 只在新版首页布局生效 |
| 设置 > `选项` > `视觉设置` > 备份与导入 | 导入 / 导出主题与动画配置 | 支持标准 JSON 和 `folia-theme://` 文本码 |

### 播放控制

| GUI 位置 | 功能 | 说明 |
| --- | --- | --- |
| 设置 > `选项` > `播放控制` > 加入队列默认位置 | `追加到末尾` / `追加到下一首` | 决定“加入队列”按钮的默认行为 |
| 设置 > `选项` > `播放控制` > `更多歌词源` | 启用 AMLLDB、QQ、酷狗等备选歌词源 | 会提升本地和 Navidrome 的歌词补全能力 |
| 设置 > `选项` > `播放控制` > `自动使用最佳歌词` | 自动比较多个来源 | 优先采用更适合逐字展示的歌词 |
| 设置 > `选项` > `播放控制` > 歌词匹配优先级 | 设定来源优先顺序 | 在多个高置信度匹配同时存在时决定偏好来源 |
| 设置 > `选项` > `播放控制` > 当前播放声卡 | 切换音频输出设备 | 桌面版优先支持；浏览器依赖 `setSinkId` |

### 存储与缓存

| GUI 位置 | 功能 | 说明 |

| 设置 > `选项` > `存储与缓存` > `已缓存歌曲数` | 查看媒体缓存规模 | 用于判断缓存是否在增长 |

### 集成设置

| GUI 位置 | 功能 | 说明 |
| --- | --- | --- |

### 桌面端设置

| GUI 位置 | 功能 | 说明 |
| --- | --- | --- |
| 设置 > `选项` > `桌面端设置` > `最小化到托盘` | 点击最小化时隐藏到托盘 | 不直接退出程序 |

## 命令面板

命令面板是播放页的隐藏中枢，默认按 `S` 打开。

### 命令类型

| 命令类型 | 典型命令 |
| --- | --- |
| 搜索 | 搜当前源、搜本地、搜 Navidrome、搜网易云 |
| 设置 | 打开帮助、打开选项、直达视觉设置、播放控制、集成设置、桌面端设置、实验室、Theme Park、歌词过滤 |
| 导航 | 回主页、回播放页、切到具体主页标签、切浏览器全屏 |
| 面板 | 打开 `封面 / 控制 / 队列 / 账号 / 本地 / Navidrome / 歌词` 标签 |
| 播放 | 播放、暂停、上一首、下一首、循环、打乱队列、自动匹配最佳歌词 |
| 视觉 | 切换 visualizer、切换背景模式、生成 AI 主题、切换透明背景、切换明暗模式 |

## 桌面版专属能力

### 遥控窗 `Folia Remote`

| GUI 位置 | 功能 | 说明 |
| --- | --- | --- |
| 桌面版 > 打开 `Folia Remote` | 独立小窗控制主播放器 | 适合主窗口全屏时切歌和录屏 |
| 遥控窗 > 顶部悬停控制区 | 切换遥控窗背景模式 | 在 `默认背景 / 封面色彩 / 透明背景` 之间轮换 |
| 遥控窗 > 顶部悬停控制区 | 固定遥控窗在最前 / 关闭遥控窗 | 只影响这个小窗自身 |
| 遥控窗 > 中间主区域 | 进度、上一首、播放/暂停、下一首、收藏 | 是最常用的遥控动作 |
| 遥控窗 > 不操作片刻后 | 歌词浮层 | 控件会暂时让位给歌词显示 |

### 遥控窗里的透明控制

| GUI 位置 | 功能 | 说明 |
| --- | --- | --- |
| 遥控窗 > `镜面/透明控制` 图标 | 打开透明控制面板 | 专门控制主窗口的透明玩法 |
| 遥控窗 > 透明控制 > `常规 / 透明` | 切换主窗口透明模式 | 与设置里的“播放页透明背景”是同一能力的遥控入口 |
| 遥控窗 > 透明控制 > `隐藏 UI / 显示 UI` | 隐藏或恢复主窗口播放页控件 | 适合录屏和直播 |
| 遥控窗 > 透明控制 > `显示边框 / 隐藏边框` | 切换主窗口边框 | 方便做无边框展示 |
| 遥控窗 > 透明控制 > `穿透` | 切换点击穿透 | 只有透明模式下才有意义 |
| 遥控窗 > 透明控制 > `置顶` | 主窗口始终在最前 | 影响主播放窗口，而不是遥控窗自身 |

### 遥控窗里的视频录制

| GUI 位置 | 功能 | 说明 |

## AI、帮助与排障入口

| GUI 位置 | 功能 | 说明 |
| --- | --- | --- |
| 设置 > `帮助` > `Show User Guide` | 新手引导 | 解释即时搜索、快捷键、命令面板、主题 |
