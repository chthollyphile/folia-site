# AI 主题

AI 主题会根据歌词内容或纯音乐标题，为当前歌曲生成一组适合 Folia 的视觉配色。

> [!TIP]
> 如果你只想知道如何使用这个AI主题功能，请前往 [AI 主题功能快速上手](/guide/basic#ai-主题功能快速上手)。

## 它会生成什么

Folia 生成的是一组“双主题”结果：

- `light`：适合明亮模式
- `dark`：适合暗色模式

每组主题通常包含：

- 主题名称
- 背景色
- 主文本色
- 强调色
- 次级文本色
- 关键词配色
- 歌词意象图标

## 支持的 AI 提供方式

目前支持两类：

- Google Gemini
- OpenAI 兼容接口

例如：

- OpenAI
- DeepSeek
- Qwen 兼容网关
- 其他兼容 `chat/completions` 的服务

## 使用前需要配置什么

### Web 版

需要在部署环境中配置：

- `VITE_AI_PROVIDER`
- `GEMINI_API_KEY` 或 `OPENAI_API_KEY`
- `OPENAI_API_URL`
- `OPENAI_API_MODEL`

### 桌面版

可以直接在应用设置中填写：

- AI 提供商
- API Key
- 兼容接口地址
- 模型名

## 什么时候能生成

通常在以下场景可用：

- 当前歌曲已有歌词
- 当前歌曲被识别为纯音乐，但拥有可用于推断氛围的标题

## 结果会如何影响界面

生成后的主题会影响：

- 背景主色
- 文本对比度
- 高亮颜色
- 歌词中的情感词着色
- 某些视觉元素的意象表达

## 建议

- 抒情、氛围感强的歌曲通常更容易得到明显差异化的主题
- AI返回主题时会有一定随机性，同一首歌多次生成可能得到不同的结果
- 如果某首歌的 AI 主题不满意，可以切回默认主题，或保存自定义主题覆盖

开发者如果要直接调用后端接口，请看 [AI 主题接口](/developer/ai-theme-api)。
